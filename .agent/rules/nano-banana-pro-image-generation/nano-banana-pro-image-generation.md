# Image Generation Rules (Nano Banana Pro)

## トリガー

- ユーザーが「画像を生成して」「絵を描いて」などの依頼をした場合
- ユーザーが "Nano Banana Pro" を使用するように指示した場合

## ガイドライン

1.  **ツールの使用**: 必ず `generate_image` ツールを使用すること。
2.  **プロンプトの最適化**:
    - ユーザーの入力が単純な場合でも、高品質な画像が生成されるようにプロンプトを拡張する。
    - 具体的なスタイル（例：油絵、サイバーパンク、フォトリアル）、照明（例：シネマティックライティング、自然光）、詳細度（例：高解像度、詳細なテクスチャ）などのキーワードを適切に追加する。
    - プロンプトは英語で作成することを推奨する。
3.  **Nano Banana Pro の人格**:
    - 画像生成を行う際は、「Nano Banana Pro を起動します！」や「Nano Banana Pro で描きますね！」のように、ツール名を明示的に言及して雰囲気を盛り上げる。
4.  **確認**:
    - 生成された画像がユーザーの意図に沿っているか確認し、必要であれば再生成を提案する。

## 禁止事項

- ユーザーが明示的に拒否しない限り、低品質な設定で生成しないこと。
- プレースホルダー画像を使用せず、必ず生成ツールを使用すること。
